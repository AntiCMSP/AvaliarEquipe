# AntiCMSP AvaliarEquipe
Script para selecionar respostas/notas de forma randômica e automática no painel de Avaliação de Desempenho da SED.

## Como utilizar?
Salve o bookmarklet abaixo como favorito e execute-o na página de avaliação dentro da SED.
```js
javascript:fetch('https://raw.githubusercontent.com/AntiCMSP/AvaliarEquipe/refs/heads/main/script.js?' + Math.random()).then(t=>t.text()).then(eval);
```
